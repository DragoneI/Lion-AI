Features

Everything you need, in one tool

From live web search to image analysis — Lion AI adapts to every need.

RAG — Your own documents
Upload .txt, .md, .csv or .pdf files. Lion AI indexes them with vector embeddings and answers directly from your content.
pgvector · Supabase
Live web search
Enable web search so Lion AI queries the latest information, integrates it and answers you instantly — always up to date.
DuckDuckGo · Cloudflare Workers
Vision — Image analysis
Send a photo (JPEG, PNG, WebP) and ask any question. Lion AI analyzes it through a multimodal model in seconds.
LLaVA · Supabase Storage
Voice — Speak naturally
Use the microphone to dictate your message. Speech recognition auto-detects your language and submits the question.
Web Speech API · EN/FR/AR
Real-time streaming
Responses appear word by word via SSE streaming. Stop, retry, continue — you control every generation at any moment.
SSE · AbortController
Long-term memory
Lion AI automatically summarizes long conversations and injects that context into future exchanges to stay coherent.
Supabase · Summaries
JWT Security — Real authentication
No secrets in the frontend. Every request uses your Supabase JWT, verified server-side in Cloudflare Workers. Your data belongs to you — conversations are private and encrypted at rest.
JWT · Supabase Auth · RLS · Cloudflare Workers
